Executivo
A Trescinco Volkswagen receberá um sistema de atendimento automatizado 24 horas via WhatsApp, integrado à API oficial do Meta — eliminando qualquer risco de bloqueio de número. A inteligência artificial conduz a conversa, identifica o interesse e a temperatura do lead, registra todos os dados na planilha existente e notifica o vendedor responsável automaticamente, seguindo o rodízio configurado entre as duas unidades.

02
Fluxo de operação

Cada mensagem percorre o seguinte caminho de forma automática e instantânea:

Etapa 01
Cliente
Envia uma mensagem para o número da Trescinco no WhatsApp.
Etapa 02
WhatsApp — API Oficial Meta
A mensagem é recebida pelo canal oficial, garantindo segurança e conformidade com as políticas do Meta.
Etapa 03
IA — Identifica contexto
A inteligência artificial lê a mensagem, identifica o contexto, a intenção do cliente e o tipo de interesse.
Etapa 04
Banco de Dados — Busca informações
Consulta o estoque, histórico do cliente e base de conhecimento para formular uma resposta precisa e personalizada.
Etapa 05
WhatsApp — API Oficial Meta
A resposta gerada pela IA é enviada ao cliente pelo canal oficial do WhatsApp.
Etapa 06
Supabase — Registra o lead
Todos os dados são gravados no banco de dados em tempo real, alimentando o dashboard e o histórico do cliente.
Etapa 07
Vendedor — Notificação WhatsApp
O vendedor alocado pelo rodízio recebe automaticamente o resumo do lead no seu WhatsApp pessoal.

09
Dashboard Online

Painel de gestão acessível pelo navegador, alimentado em tempo real pelo Supabase. Organizado em 5 páginas independentes.

◎
Visão Geral — KPIs em tempo real
Total de leads (dia / semana / mês), taxa de conversão geral, distribuição por temperatura, origem dos leads (WhatsApp / Instagram / Facebook / Usado Fácil) e modalidade mais frequente.
◎
Leads ao Vivo
Tabela atualizada em tempo real com Cliente, Telefone, Origem, Interesse, Modalidade, Modelo, Status, Vendedor, Data de entrada e Última interação. Filtros por loja, status e período.
◎
Funil de Vendas
Representação visual em formato de funil com as 4 etapas: Frio → Morno → Quente → Vendido. Exibe volume de leads e percentual de conversão entre cada etapa.
◎
Ranking de Vendedores
Classificação por período (dia / semana / mês) com leads recebidos, leads convertidos e taxa de conversão individual. Separado por Loja 1 e Loja 2.
◎
Gestão de Vendedores
Página administrativa para cadastrar, editar, ativar e desativar vendedores. Exibe posição atual no rodízio, loja associada e número de WhatsApp. Alterações refletem imediatamente no sistema.

O que precisamos da Trescinco

Para iniciar a implementação, os itens abaixo são necessários:

Acesso ao Meta Business Manager para configuração do webhook do WhatsApp Business API
Criação e acesso ao projeto Supabase — banco de dados que substituirá o Google Sheets
Base de estoque de veículos disponíveis — pode ser uma aba na própria planilha, atualizada pela equipe
Números de WhatsApp de todos os vendedores da Loja 1 para cadastro no sistema de rodízio
Definição do tom de voz da IA — formal, descontraído ou intermediário
Confirmação de quais vendedores pertencem a cada loja para correta configuração do rodízio
